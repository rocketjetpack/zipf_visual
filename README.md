# Zipf's Law Explorer

This repository contains a static interactive page that visualizes word frequency and explains Zipf's Law.

## Local Preview

Open `index.html` in a browser, or serve the directory with any static file server.

## GitHub Pages

The repository includes a GitHub Actions workflow at `.github/workflows/deploy-pages.yml`.

After you push this repo to GitHub:

1. Go to the repository settings.
2. Enable GitHub Pages using `GitHub Actions` as the source.
3. Push to `main` to trigger deployment.
